This function clears the current menu and initializes two static pointers to arrays\+: users and actions. It reads the tables from files USERFILENAME and ACTIONFILENAME, and if users is null, it allocates memory for a \mbox{\hyperlink{structUser}{User}} struct and assigns the given user to the first element. Finally, it calls the \mbox{\hyperlink{TimeTrackingMenu_8c_a4058239e2263d26b2c1c8dc953596297}{run\+Program()}} function with users and actions as arguments to run the Time Tracking Menu.

{\footnotesize\ttfamily void initial\+Ncurses (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Initializes ncurses and sets it up for use in the program. 

This function initializes the ncurses library by calling the initscr() function. It also enables cbreak mode to make button presses immediately available to the program. Additionally, it disables automatic printing of typed characters using noecho(). Finally, it sets the stdscr screen up to receive special keys by calling the keypad() function with TRUE as the argument. \begin{DoxyNote}{Note}
This function should be called at the beginning of any program that uses ncurses.
